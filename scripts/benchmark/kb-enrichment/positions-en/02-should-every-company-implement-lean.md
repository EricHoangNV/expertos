# Should every company implement Lean?

No. A company should not adopt Lean simply because it is popular or because competitors use it. Lean is a means to remove waste and improve the flow of value to the customer — it is not a universal mandate, and applying it reflexively wastes effort.

The right question is not "should we do Lean?" but "what problem are we solving, and what method fits it?". Match the approach to the actual constraint and the customer value at stake. Adopt a tool because the problem calls for it, never because it is fashionable.
