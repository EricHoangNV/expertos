# Held-out question sets

The 100-question UAT set has a validity limit: the KB enrichment (framework docs, position
docs) was authored *while looking at these questions*, so scores on them are an upper bound.
Two held-out mechanisms exist to check whether the gains generalize:

## 1. Paraphrase set (available now) — `heldout-paraphrase.{en,vi}.json`

Generated by `make-paraphrase-set.cjs`: 25 questions per language (all 15 position-target
archetypes + 10 controls), rewritten with substantially different wording but identical
meaning. Gold answers unchanged. This tests **phrasing overfit** — whether retrieval only
works because the enrichment docs mirror the exact question wording.

```bash
node scripts/benchmark/make-paraphrase-set.cjs                  # regenerate (LLM paraphrase)
node scripts/benchmark/run.cjs   --set heldout-paraphrase --run-id para-<date>
node scripts/benchmark/score.cjs --run-id para-<date> --judge-model gpt-4o
```

Read it by group: `heldout_group: "position-target"` vs `"control"`. Targets cratering while
controls hold = position docs are overfit to phrasing.

**Limits:** same topics, same gold answers — it cannot detect topic-level teaching-to-the-test.

## 2. True held-out set (needs the expert) — `heldout-expert.{en,vi}.json` (not yet created)

The real generalization check needs 20–30 **new** Q&A pairs authored by the expert (John
Ngo) that nobody consulted while building the enrichment. Request format — same columns as
the original sheet:

- Question (VN) / Answer (VN) / Question (EN) / Answer (EN)
- GOLD metadata table (Category, Level, Persona, Difficulty, Trap Question, Frameworks)
- Mix: ~half on topics the existing 100 already cover (tests generalization within covered
  ground), ~half on adjacent topics not in the current set (tests coverage honestly)
- Keep them sealed: do not share with anyone doing KB enrichment before the first run

Convert with `extract.py <new-sheet>.xlsx` (adjust output names to `heldout-expert.*`), then
run with `--set heldout-expert`. Compare against the main set: if the main set scores ~70 and
the expert held-out scores ~59 (the pre-enrichment baseline), the enrichment memorized rather
than generalized; if both land in the same band, the KB genuinely improved.
